Esercizio n. 2 - Magazzino

Creare una classe Prodotto che ha le seguenti proprietà: codice, nome, descrizione, prezzo, giacenza(quantità di pezzi a magazzino) e realizzare una classe con Main in cui caricare un certo numero di prodotti (es. 10) in un array o lista. 

Si richiedono le seguenti funzionalità: 

visualizzare il prodotto con prezzo maggiore
cercare un prodotto di cui si conosce il codice
stabilire quanto è il valore del magazzino (per ogni prodotto, il valore = prezzo del prodotto * quantità)
elencare i prodotti che sono in scorta (giacenza [1-9])
elencare i prodotti esauriti (giacenza=0)
Creare un piccolo menu testuale per eseguire le varie operazioni richieste.
